Welcome to LogiCAD!


LogiCAD is a digital circuit Computer-Aided Design application. As you dive into our software you may notice several tools and menus at your disposal, so lets take a look at what we have to offer:


Top Menu Bar:

-Help: Drop down menu of the following:

    About:  Information about the software and button features.

    Shortcut key to the corresponding gate. Escape key can be used to exit out your current tool.



Gate and Operations Menu Bar:

-AND Gate Button:  An AND gate will appear on the grid workspace for a user to be able to work with. This logic gate must be given two inputs and will provide an output.
-OR Gate Button:  An OR gate will appear on the grid workspace for a user to be able to work with. This logic gate must be given two inputs and will provide an output.
-NOT Gate Button:  A NOT gate will appear on the grid workspace for a user to be able to work with. This logic gate must be given one input and will provide an output.
-XOR Gate Button:  An XOR gate will appear on the grid workspace for a user to be able to work with. This logic gate must be given two inputs and will provide an output.
-NAND Gate Button:  A NAND gate will appear on the grid workspace for a user to be able to work with. This logic gate must be given two inputs and will provide an output.
-NOR Gate Button:  A NOR gate will appear on the grid workspace for a user to be able to work with. This logic gate must be given two inputs and will provide an output.
-XNOR Gate Button:  A XNOR gate will appear on the grid workspace for a user to be able to work with. This logic gate must be given two inputs and will provide an output.
-Input Gate Button:  An Input circuit element will appear in the workspace. It will appear, by default, with a 0 inside signifying the signal it will produce. It can be toggled to a 1 using the Toggle Input button.
-Output Gate Button:  An Output circuit element will appear in the workspace. It will appear, by default, with nothing inside of it signifying that no signals have been propagated to it. It will change to a 0 or 1
when the Evaluate button is clicked and there are no errors. If errors are present, an 'X' will appear in the output.
-Delete Button:  The delete button allows the user to delete a single circuit element in the workspace. When deleted, any connections to that circuit element will be erased, including the wires.
-Connect Button:  The connect button allows the user to connect two circuit elements. To make a proper connection the user must first click connect and then select an input
or source gate (gate whose output does not have a connection). Then the user should select the destination gate (gate that has a free input). When both have been selected a connection
will have been made and a wire will be drawn between them. NOTE: To prevent wires from drawing over other circuit elements or wires the circuit elements must be laid out in a manner
such that the path (using 90 degree angles) has no circuit elements.
-Toggle Input:  The toggle input button is specifically used to toggle the Input circuit element to be either a 0 or a 1. This will be the value of the signal that propagates through the circuit.
-Cancel Button:  The cancel button will invalidate any button selection or current operation.
-Evaluate Button:  The Evaluate button will propagate the signal(s) through the circuit and display the output in the output circuit elements as a 0 or 1. For the evaluate button to trigger the
propagation there must be no dangling wires and each gate must be fulle connected(no free inputs or outputs). If there are, an error window will popup and the outputs are set to 'X', signifying that
there are errors in the workspace.

-This gate menu bar is able to be moved around. By clicking the very left of the bar, a user can drag the menu bar all around and place it wherever one wishes.
If a user decides to even take the menu bar outside the window they may do so. When they click the exit button the bar will go back to where it last was.



Grid Workspace:

-This is the area where circuit elements are placed. Circuit elements can be placed freely in the workspace. As more elements are added the workspace will expand to the right and down as needed.
To navigate the workspace move the scroll bars that appear on the bottom and right of the workspace.

Making Circuits:

-Making proper circuits in LogiCAD involves following a couple of contraints.
  1. When making conections click the source gate first and the target gate second.
  2. Inputs can not be the target of a connection.
  3. Outputs can not be the source of a connection.
  4. Feedback loops are not allowed. Meaning that the target of a connection can not be a gate that is an input to the source.
